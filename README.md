CSanquer TwigExtraBundle
========================

[![Build Status](https://travis-ci.org/csanquer/TwigExtraBundle.png)](https://travis-ci.org/csanquer/TwigExtraBundle)

Requirements
------------

* PHP >= 5.3.3
* Twig >= 1.12
* Symfony >= 2.1
